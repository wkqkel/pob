## **💻 설치 및 실행 방법**
<br />
1. yarn 설치

```
 npm i yarn
```

2. 레포지토리 클론

```
git clone git@https://github.com/wkqkel/portfolio.git
```

3. dependencies 설치

```
yarn install
```

4. 실행

```
yarn start
```
